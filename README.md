# Cake-ordering-platform
<p align="justify">
This Cake Shop Website is developed using PHP, HTML, CSS, JavaScript, and MySQL. It allows users to view different cake varieties and chocolates with details such as image, name, price, and quantity. Users can register, log in, add cakes to the cart, and place orders through a checkout system. The website is fully responsive and all data is dynamically managed using a MySQL database, making it user-friendly and efficient.
</p>
